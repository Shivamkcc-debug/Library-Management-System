Library Management System

A web-based Library Management System developed using Flask and SQLite.

## Features

- Admin Login
- Add Books
- Manage Books
- Issue Books
- Return Books
- Due Date Tracking
- SQLite Database Integration

## Tech Stack

- Python
- Flask
- SQLite
- HTML
- CSS

## Installation

```bash
pip install -r requirements.txt
python app.py
